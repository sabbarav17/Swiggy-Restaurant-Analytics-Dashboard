# 🍽️ Swiggy Restaurant Analytics Dashboard

## 📌 Project Overview

This project presents an end-to-end Business Intelligence solution using SQL Server and Power BI to analyse restaurant performance, customer ratings, popular dishes, city-wise trends, and category insights from Swiggy restaurant data.

The project demonstrates a real-world analytics workflow, including data cleaning, SQL analysis, KPI development, dashboard design, and business recommendations.

---

## 🎯 Business Objectives

- Analyse restaurant performance
- Identify top-performing cities
- Evaluate customer ratings
- Discover popular dishes
- Analyse category-wise performance
- Support data-driven business decisions

---

## 🛠️ Tools Used

- SQL Server
- Power BI
- Microsoft Excel
- Git & GitHub
