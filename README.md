# 🍽️ Swiggy Restaurant Analytics Dashboard

## 📌 Project Overview

This project presents an end-to-end Business Intelligence solution using **SQL Server** and **Power BI** to analyse restaurant performance, customer ratings, menu popularity, city-wise trends, and food category performance using Swiggy restaurant data.

The project demonstrates a complete analytics workflow, including data cleaning, SQL analysis, KPI development, interactive dashboard creation, and business recommendations to support data-driven decision-making.

---

## 🎯 Business Objectives

- Analyse overall restaurant performance
- Identify high-performing restaurants
- Evaluate city-wise business performance
- Discover popular dishes and food categories
- Measure customer engagement through ratings
- Support strategic business decisions

---

## 📊 Key Performance Indicators (KPIs)

- Total Restaurants
- Total Cities
- Total States
- Total Dishes
- Average Rating
- Total Customer Ratings
- Restaurant Performance Score
- City Performance Score
